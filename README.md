# الساقية - React Website

React version of the Al-Saqiah food company website.

## Features

- **React 19** with TypeScript
- **React Router** for navigation
- **Tailwind CSS** for styling
- **RTL (Right-to-Left)** support for Arabic
- Responsive design
- Modern UI with gradient themes

## Pages

- `/` - Home page with hero section, featured products, news, and brands
- `/about` - About us page
- `/products` - Products gallery
- `/news` - News and blog section
- `/contact` - Contact form
- `/sustainability` - Sustainability information

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
  components/     # Reusable components (Header, Footer, Layout)
  pages/          # Page components (Home, About, Products, etc.)
  index.css       # Global styles with Tailwind
```

## Assets

- `public/logo.svg` - Company logo
- `public/brands/` - Brand logos
- `public/Produit/` - Product images
